#  ShopZone - E-commerce React App

##  Project Overview
ShopZone is a simple e-commerce web application built using React. It allows users to browse products, view details, and add/remove items from the cart.

---
##  Features
-  Home page with banner
-  Add to Cart functionality
-  Remove from Cart
-  Cart persistence using LocalStorage
-  Shop page with product listing
-  Product cards with clean UI
-  Routing using React Router
---
## 🛠️ Tech Stack
- React.js
- CSS
- JavaScript (ES6)
- React Router
